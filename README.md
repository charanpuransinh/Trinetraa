# 🔱 TRISHUL PRO - ADVANCED ALGO TRADING SYSTEM

TRISHUL PRO is a high-performance, news-based algorithmic trading application integrated with India's leading brokers (Angel One & Dhan). It features a real-time sentiment scanner (Trishul Scanner V4) that processes live news to generate aggressive trading signals.

## 🚀 KEY FEATURES
* **Trishul Scanner V4:** Advanced news-based logic with Duplicate Filters and News Memory.
* **Multi-Broker Sync:** Seamless integration with Angel One (P245643) and Dhan (1100345087).
* **Real-time News Feed:** Live data aggregation from MoneyControl, Google News, and Broker APIs.
* **Aggressive Alerts:** Custom bell notifications and color-coded signals based on news intensity.
* **Auto-Deployment:** Optimized for AWS EC2 server environments.

## 📂 PROJECT STRUCTURE
* `trishul_login.html` - Main Entry Point & Authentication.
* `trishul_home.html` - Professional Dashboard with Live P&L.
* `trishul_admin.html` - Master Control Panel & API Configuration.
* `trishul_signals.html` - Real-time Signal Monitoring.
* `trishul_charts.html` - Advanced Charting Interface.
* `scripts/scanner-v4.js` - The Core Trading Logic (V4 Engine).

## 🛠️ SETUP & INSTALLATION
1. Clone the repository to your AWS EC2 instance.
2. Configure your Angel One and Dhan API keys in `trishul_admin.html`.
3. Launch `trishul_login.html` to start the application.
4. Ensure `scanner-v4.js` is active for live news processing.
